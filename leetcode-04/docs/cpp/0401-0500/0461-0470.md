## 461. Hamming Distance [$\star$](https://leetcode.com/problems/hamming-distance)

```cpp
class Solution {
 public:
  int hammingDistance(int x, int y) {
    int ans = 0;

    while (x || y) {
      ans += (x % 2) ^ (y % 2);
      x /= 2;
      y /= 2;
    }

    return ans;
  }
};
```
