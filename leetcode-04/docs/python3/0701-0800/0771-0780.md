## 775. Global and Local Inversions [$\star\star$](https://leetcode.com/problems/global-and-local-inversions)

```python
class Solution:
    def isIdealPermutation(self, A: List[int]) -> bool:
        for i, a in enumerate(A):
            if abs(a - i) > 1:
                return False

        return True
```
